README.TXT - CS344 HOMEWORK 3 - smallsh

To execute smallsh, perform the following steps:

1.  Unzip the contents of "smallsh_BenjaminFridkis.zip" to a directory of 
		your choice.

2.  Navigate to the directory chosen in step 1 (in your Linux console), 
		and type "make" (without quotes).

3.  Type "./smallsh" (without quotes) to run the program. 

4.  When finished with smallsh, type "make clean" (without quotes) to
		delete the smallsh executable file and all associated object (.o)
		files.
